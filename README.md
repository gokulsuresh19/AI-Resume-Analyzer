# AI Resume Analyzer

## Overview
AI Resume Analyzer is a Streamlit-based application that compares resumes with job descriptions and provides a match score.

## Features
- Upload PDF resumes
- Job description matching
- Skill extraction
- Missing skill identification
- Resume match percentage

## Technologies Used
- Python
- Streamlit
- PyPDF2

## How to Run
pip install -r requirements.txt
streamlit run test.py
